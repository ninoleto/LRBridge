Bitfocus Companion HTTP Builder

For Bitfocus Companion with the Generic: HTTP Requests plugin/add-on.

Security and network exposure

LRBridge has no login or built-in authentication. Use this API only on a trusted private LAN or a restricted Tailscale network. 127.0.0.1 works only when Companion and LRBridge run on the same computer. For another computer, use the LRBridge computer's stable LAN or Tailscale address and scope Windows Firewall access to the Companion computer where practical. Never publicly forward ports 17890, 17891, or 17892, and do not use Tailscale Funnel, a public reverse proxy, or a public cloud tunnel.

How to use this page with Bitfocus Companion

  1. Open Bitfocus Companion.
  2. Go to Connections.
  3. Add the connection/plugin called Generic: HTTP Requests.
  4. In that connection, set the Base URL to your LRBridge address. You can use the Base URL helper below to copy it.
  5. Save/apply the connection.
  6. Go to your Companion button page.
  7. Create or edit a button.
  8. Add a press action called http: GET.
  9. In the URI field, paste only the command from this page.

The Base URL is set once in the Generic: HTTP Requests connection. Each button only needs the URI.

Example URI: /adjust?slider=Exposure&amount=-1

Copy path or copy full URL?

Copy path is for Bitfocus Companion button actions. Paste it into the URI field of an http: GET action.

Copy full URL is for PowerShell, browser testing, curl, scripts, or other HTTP-capable devices/apps. PowerShell can send a command directly to Lightroom through LRBridge.

curl.exe "http://127.0.0.1:17891/adjust?slider=Exposure&amount=1"

1. Base URL helper

This section only helps you generate the Base URL for Bitfocus Companion or PowerShell.

If Bitfocus Companion runs on the same PC as LRBridge, use localhost. If Bitfocus Companion runs on another PC, type the LAN IP of the LRBridge PC.

All Copy full URL buttons on this page are generated from this Base URL.

Current Base URL:

http://127.0.0.1:17891

2. Step size

Step size controls how strong each slider button is. Default is 1.

If you type 4, slider buttons will generate: -4, Reset, and +4.

Reset is always included for every slider.

Current step size: 1
