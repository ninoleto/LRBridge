## Confirmed failures

For every failure below, the HTTP request was accepted and queued, but Lightroom Classic 15.3 produced no visible result:

1. `application.view / people`
2. `application.view / develop_loupe`
3. `application.view / develop_before_after_vert`
4. `application.action / toggle_zoom`
5. `application.action / zoom_in`
6. `application.action / zoom_out`
7. `application.action / fullscreen_hide_panels`
8. `application.action / cycle_loupe_info`
9. `application.secondary_view / slideshow`

For `people`, `LrApplicationView.showView("people")` is explicitly documented in the local Lightroom Classic 15.3 SDK. It produced no visible change twice in Library Grid, while manually pressing O opened People immediately in the same catalog and UI state. No prerequisite is documented.

These outcomes demonstrate accepted transport and queue processing, not successful SDK execution.
