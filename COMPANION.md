## Reset all mapped sliders

One Companion button can reset all currently mapped sliders:

```text
/reset-all
```

This queues reset commands for all supported sliders:

```text
Exposure
Contrast
Highlights
Shadows
Whites
Blacks
Temperature
Tint
Texture
Clarity
Dehaze
Vibrance
Saturation
Sharpness
LuminanceNR
ColorNR
```

Suggested button label:

```text
RESET ALL
```
